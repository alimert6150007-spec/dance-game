## 🎮 Oyun Özellikleri

- 🕺💃 **Karakter Seçimi** — B-Boy veya B-Girl seç
- 🎵 **5 Farklı Şarkı** — Boom Bap, Trap, Funk, Lo-Fi, Hardcore
- 🎯 **3 Zorluk Seviyesi** — Kolay, Orta, Zor
- ⭐ **Puanlama** — Perfect, Great, Good, Miss
- 🔥 **Combo Sistemi** — Ardışık isabetlerle puan çarpanı
- 🏆 **Derece Sistemi** — S, A, B, C, D, F
- 💥 **Breakdance Animasyonları** — Windmill, Freeze ve daha fazlası
- 🔊 **Prosedürel Müzik** — Web Audio API ile gerçek zamanlı ses üretimi
- ✨ **Neon/Cyberpunk Tasarım**

## 🎹 Kontroller

| Tuş | Yön |
|-----|-----|
| ← veya D | Sol |
| ↓ veya F | Aşağı |
| ↑ veya J | Yukarı |
| → veya K | Sağ |

## 🚀 Nasıl Oynanır?

1. `index.html` dosyasını tarayıcıda açın
2. Karakterinizi seçin (B-Boy / B-Girl)
3. Şarkınızı seçin
4. Zorluk seviyesini belirleyin
5. **BATTLE BAŞLAT** butonuna basın
6. Düşen oklara zamanında basın!
